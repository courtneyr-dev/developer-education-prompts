# Getting Started with Web Development

## Purpose
Help beginners understand the fundamental concepts of web development and create their first webpage.

## Target Audience
- Skill Level: Beginner
- Prerequisites: Basic computer literacy

## Prompt

```
I'm new to web development and want to learn the basics. Can you explain:
1. What HTML, CSS, and JavaScript are and how they work together?
2. Walk me through creating a simple "Hello World" webpage
3. Show me how to add basic styling with CSS
4. Explain how to add interactivity with JavaScript

Please use simple language and include code examples with explanations.
```

## Example Usage
Use this prompt when starting your web development journey to get a comprehensive overview of the basic building blocks.

## Expected Outcome
- Clear explanation of HTML, CSS, and JavaScript roles
- Working code for a simple interactive webpage
- Understanding of how these technologies interact

## Tips
- Try typing out the code examples yourself rather than copy-pasting
- Experiment with changing values to see what happens
- Ask follow-up questions about anything you don't understand
