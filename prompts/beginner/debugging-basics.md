# Debugging Basics for Web Development

## Purpose
Help beginners understand debugging techniques and tools for finding and fixing code issues.

## Target Audience
- Skill Level: Beginner
- Prerequisites: Basic HTML, CSS, and JavaScript knowledge

## Prompt

```
I'm struggling to find and fix bugs in my code. Can you teach me:
1. How to use browser developer tools (console, debugger, network tab)
2. Common JavaScript errors and how to read error messages
3. Basic debugging techniques (console.log, breakpoints, step-through)
4. How to debug CSS layout issues
5. Tips for preventing common beginner mistakes

Please provide practical examples of debugging real problems.
```

## Example Usage
Use this prompt when you encounter errors or unexpected behavior in your web applications.

## Expected Outcome
- Understanding of browser developer tools
- Ability to read and interpret error messages
- Practical debugging workflow
- Prevention strategies

## Tips
- Read error messages carefully - they often tell you exactly what's wrong
- Use console.log liberally while learning
- Learn keyboard shortcuts for dev tools
- Practice on simple examples first
- Don't be afraid to break things while learning
