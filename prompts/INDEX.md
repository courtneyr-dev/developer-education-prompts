## How to Choose a Prompt

1. **Identify your skill level**: Are you a beginner, intermediate, or advanced developer?
2. **Find your topic**: Browse the table above for topics that match your learning goals
3. **Review the prompt**: Open the markdown file to see prerequisites and expected outcomes
4. **Customize if needed**: Adjust the prompt to fit your specific situation
5. **Use with AI**: Copy and paste into your preferred AI assistant

## Suggesting New Prompts

If you have ideas for prompts that would be valuable to the community:

1. Check if a similar prompt already exists
2. Use the template in `templates/prompt-template.md`
3. Follow the contribution guidelines in the main README
4. Submit a pull request
